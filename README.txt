
== Devel Admin Menu UX

Adds all Devel menu links to the the main Navigation menu. Designed to use together
with Admin menu, so the all Devel links could be accessed from everywhere.
Leaves original Devel menu untouched.

_________________________________________________________________________________________

== Requires

* Devel - http://drupal.org/project/devel
* Admin menu (optional) - http://drupal.org/project/admin_menu
